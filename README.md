# 🏥 greenlife

## Healthcare Management System (Personal Project)

A full-stack healthcare management application built for learning purposes.  
The system manages patients, doctors, and appointments using a **Flutter (Dart) mobile app**, a **Node.js backend API**, and a **PostgreSQL database**.

---

## 📌 Overview

This project simulates basic healthcare workflows such as:

- Patient registration and login  
- Doctor management  
- Appointment scheduling  
- Basic medical record tracking  

It is designed to learn full-stack mobile development with REST APIs and relational databases.

---

## ✨ Features

- 👤 Patient registration & authentication  
- 👨‍⚕️ Doctor profiles & specialization management  
- 📅 Appointment booking system  
- 📝 Medical records storage  
- 🔐 Secure login using JWT authentication  
- 🧑 Role-based access (Admin / Doctor / Patient)  
- 📊 Simple and clean dashboard UI  

---
